# School ERP

Standalone school and preschool ERP demo repository.

This repo is intentionally separated from the SrS Logics website and now contains only the ERP demo app files.

## What Is Included

- `octominds-preschool-demo/`
  Standalone founder-facing preschool ERP demo
- `index.html`
  Root redirect into the demo

## Demo Features

- founder control room
- admissions workflows
- centre operations
- parent app experience
- fees and collections
- child progress tracking
- installable PWA setup

## Local Preview

```bash
python3 -m http.server 4173
```
